## Training and testing

Yay! Here's the final step. It remains to train our model and make sure it works properly.

### Task
To do this, fill in the placeholders and run the script. Check `policy` bar plot.

### ⚠ Important note
The validation script does not re-run the training! This is done so that the validation does not take too long.

Therefore, to check a task, first train the model (by running the current file), and after training, press **Check**.  
The check script uses the model with index `args['num_iterations'] - 1`.
