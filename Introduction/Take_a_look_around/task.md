The course is logically divided into lessons in which we will go from creating a simple board game to training a model capable of playing complex board games.

### Trial course
#### TicTacToe
Creating a field and methods to play a TicTacToe. The result will be a game that you can play with a friend.

#### MCTS
Implementation of the Monte Carlo Tree Search (MCTS) algorithm. As a result, MCTS will be the second TicTacToe player. 

#### ResNetEstimator
Here we will start implementing the ResNet model for next move prediction and position estimation and make sure it can interact with our game field.

### Full course 
#### AlphaMCTS
In this step, we will change the implementation of MCTS algorithm to add the policy and value from the neural network.

#### AlphaZero
Here we finally implement methods to simulate a game where the AI plays against itself and to train the neural network model using the data collected from self-play. 

#### ConnectFour
In this step, we will develop a ConnectFour game field and see how easily we can train an existing model for another game!

#### DotsAndBoxes
In this last lesson, we will implement the Dots and Boxes game and add a graphical interface to it. Combined with the model trained for this game, we will get a complete application that can play against you. Will you try to win it?
